Title
Sales Performance Analysis Dashboard
Project Objective
The main goal of this project is to analyze sales data and understand sales trends, revenue growth, and product performance. This helps the business make better decisions.
Tools Used
•	Excel
•	Power BI
•	SQL (for data extraction)
Data Understanding
The dataset contains:
•	Order Date
•	Product Name
•	Category
•	Sales
•	Quantity
•	Profit
•	Region
 Data Cleaning & Preparation
•	Removed duplicate records
•	Handled missing values
•	Changed date formats
•	Created new calculated columns
Analysis Performed
•	Total Sales and Profit calculation
•	Monthly and yearly sales trends
•	Top-performing products
•	Region-wise sales comparison
Dashboard Insights
•	Sales increased during festival months
•	Some products have high sales but low profit
•	South and West regions perform better
Business Impact
•	Helps management track performance
•	Identifies low-profit products
•	Supports data-driven decision making
Conclusion
This project improved my skills in data analysis, dashboard creation, and business understanding. It shows how data can help improve sales strategy.
